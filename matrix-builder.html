Overview of Properties

Commutative The order of the strings does not matter
Complement The complement of a string is the string with all 1s replaced with 0s and vice versa
Reverse The reverse of a string is the string with its characters in reverse order
Slice and -fix If two strings have a common -fix, replacing the -fix with another string of equal length will result in the same LCS length
For more information and proof of these properties, read the LCS Properties document.
